部署说明

将此页面部署到 GitHub Pages

  1. 将 `index.html` 与 `style.css` 上传到 GitHub 仓库根目录，或放入 `docs/` 目录。
  2. 进入仓库设置：Settings → Pages，选择 `main` 分支和 `root` 或 `docs/` 目录。
  3. 保存后等待几分钟，即可生成 GitHub Pages 地址。
  4. 在项目卡片中替换成真实 GitHub 仓库地址。

如果你需要，我还可以继续帮你把这套页面改成 `docs/` 结构、写成 Jekyll 模板，或添加项目截图和详细页面。
