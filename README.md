# GitHub Pages 项目展示框架

这是一个可直接复制到 GitHub Pages 的静态项目展示模板，包含一个简洁的 `index.html` 和配套样式 `style.css`。

## 使用方法

1. 将整个 `github-pages-portfolio` 文件夹上传到你的 GitHub 仓库。
2. 如果希望直接发布到仓库根目录，可把 `index.html` 和 `style.css` 放在仓库根目录。
3. 进入 GitHub 仓库设置：Settings → Pages。
4. 选择 `main` 分支和 `root` 或 `docs/` 目录作为发布源。
5. 保存后等待几分钟，即可访问 GitHub Pages 页面。

## 推荐改动

- 将项目卡片中的“仓库链接（请替换）”替换成你自己的 GitHub 仓库地址。
- 如果你想展示更多项目，可以复制 `article.project-card` 模块并更新内容。
- 如果你想把项目拆成多页，可新增 `project-xxx.html` 页面并在卡片里链接它们。

## 备注

- 该页面结构适合用于个人主页或作品集展示。
- 如果你要继续美化页面，我可以帮你增加项目截图、图标、或者制作更完整的 `docs/` 结构。
